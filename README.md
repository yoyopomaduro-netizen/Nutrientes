# NutriLens AI 🥗

**NutriLens AI** — Fotografía tu plato y obtén proteínas, carbos y grasas al instante. La IA aprende de tus correcciones y adapta cada consejo a tu meta: bajar peso, ganar músculo o comer mejor.

## Características

- 📷 Escaneo con cámara en tiempo real o desde galería
- 🧠 Análisis nutricional con IA (Claude de Anthropic)
- 🎯 Consejos personalizados según tu objetivo
- 🔄 Sistema de corrección que aprende de tus errores
- 📋 Historial de platos escaneados
- 📱 Instalable como app (PWA)

## Cómo usar

1. Abre la app en tu navegador
2. Ve a **Mis metas** y configura tu objetivo
3. Vuelve a **Escanear**, toma una foto o sube una imagen
4. Toca **Analizar plato** y obtén tus macros

## Tecnologías

- HTML + CSS + JavaScript puro
- API de Claude (Anthropic) para visión computacional
- LocalStorage para historial y correcciones
- PWA instalable sin app store
